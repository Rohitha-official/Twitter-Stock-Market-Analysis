# Twitter-Stock-Market-Analysis

## Project Overview
This project analyzes Twitter's historical stock market performance (2018–2024) using Python and Power BI. It explores stock price trends, trading volume, and key market insights through data visualization and an interactive dashboard.

![Twitter Stock Market Analysis Dashboard]

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Power BI

## Dataset
- Historical Twitter (TWTR) stock data (2018–2024)
- Features include Date, Open, High, Low, Close, Adjusted Close, and Volume.

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Stock Price Trend Analysis
5. Trading Volume Analysis
6. High vs Low Price Comparison
7. Interactive Dashboard Development using Power BI

## Key Insights
- Twitter stock showed significant price volatility during the study period.
- Highest Closing Price: **77.63 USD**
- Lowest Closing Price: **14.01 USD**
- Average Closing Price: **36.00 USD**
- Total Trading Volume: **49 Billion+**
